The plugin includes full support for WordPress Dashicons, allowing you to add visual icons to every menu item for better user experience and faster navigation.

## Overview

Icons appear to the left of menu item titles and are rendered using the WordPress Dashicons font library, which is included with every WordPress installation.

<Info>
  Dashicons are vector icons that scale perfectly at any size and automatically match your site's color scheme.
</Info>

## Adding Icons to Menu Items

<Steps>
  <Step title="Open Item Editor">
    Click the edit button (✏️) on any menu item, or create a new item.
  </Step>
  
  <Step title="Find Icon Field">
    Scroll to the **Icon** field in the modal.
  </Step>
  
  <Step title="Choose Icon">
    Either:
    - Click a preset icon from the grid picker
    - Manually enter the icon class (e.g., `dashicons dashicons-star-filled`)
  </Step>
  
  <Step title="Preview Updates">
    The icon preview next to the input field updates in real-time.
  </Step>
  
  <Step title="Save">
    Click **Save Item** to apply the icon.
  </Step>
</Steps>

## Icon Class Format

Dashicons use a two-part CSS class structure:

```css
dashicons dashicons-{icon-name}
```

**Examples:**
```text
dashicons dashicons-dashboard
dashicons dashicons-cart
dashicons dashicons-star-filled
dashicons dashicons-admin-users
```

<Warning>
  Both classes are required. Using only `dashicons-cart` without `dashicons` won't render the icon.
</Warning>

## Frontend Icon Rendering

Icons are rendered via JavaScript on the frontend:

```javascript
// Frontend icon injection (assets/js/frontend.js)
$.each(wmabFront.items, function(key, config) {
    var $link = $('.woocommerce-MyAccount-navigation-link--' + key + ' a');
    
    // Add icon if specified
    if (config.icon) {
        $link.prepend('<i class="' + config.icon + '"></i> ');
    }
});
```

The Dashicons font is enqueued automatically:

```php
// Dashicons enqueue (class-wmab-frontend.php:124)
wp_enqueue_style( 'dashicons' );
```

## Styling Icons

You can customize icon appearance with CSS:

### Change Icon Color

```css
/* All menu icons */
.woocommerce-MyAccount-navigation a i.dashicons {
    color: #0073aa;
}

/* Active item icon */
.woocommerce-MyAccount-navigation li.is-active a i.dashicons {
    color: #ff6600;
}

/* Specific item icon */
.woocommerce-MyAccount-navigation-link--orders a i.dashicons {
    color: #46b450;
}
```

### Adjust Icon Size

```css
.woocommerce-MyAccount-navigation a i.dashicons {
    font-size: 20px;
    width: 20px;
    height: 20px;
}
```

### Icon Spacing

```css
.woocommerce-MyAccount-navigation a i.dashicons {
    margin-right: 8px;
}
```

### Icon Animations

```css
/* Rotate icon on hover */
.woocommerce-MyAccount-navigation a:hover i.dashicons-admin-settings {
    transform: rotate(90deg);
    transition: transform 0.3s ease;
}

/* Pulse effect */
@keyframes pulse {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.1); }
}

.woocommerce-MyAccount-navigation a:hover i.dashicons {
    animation: pulse 0.5s ease;
}
```

## Custom Icon Fonts

<Info>
As of version 1.3.0, the plugin exclusively uses WordPress Dashicons (bundled with core). Custom icon fonts like Font Awesome are not included to keep the plugin lightweight (~95% size reduction).
</Info>

If you need additional icons beyond Dashicons, you can enqueue your own icon font:

### Custom Icon Font Example

```php
// Enqueue custom icon font on My Account page
add_action( 'wp_enqueue_scripts', function() {
    if ( is_account_page() ) {
        wp_enqueue_style( 'my-custom-icons', get_stylesheet_directory_uri() . '/css/custom-icons.css' );
    }
}, 20 );
```

Then use your custom icon classes in the icon field:

```text
custom-icon-star
custom-icon-cart
custom-icon-heart
```

<Warning>
Custom icon fonts require additional CSS to align properly with menu text and may increase page load time. Test thoroughly before deploying.
</Warning>

## Troubleshooting

### Icon Not Displaying

**Possible causes:**
1. **Missing `dashicons` prefix** — Ensure you use `dashicons dashicons-{name}`
2. **Dashicons not enqueued** — The plugin auto-enqueues on My Account pages
3. **Theme CSS conflict** — Check if theme CSS is hiding icons

**Solution:**
```php
// Force enqueue Dashicons everywhere (if needed)
add_action( 'wp_enqueue_scripts', function() {
    wp_enqueue_style( 'dashicons' );
});
```

### Icon Appears as Square/Box

**Cause:** The Dashicons font file isn't loading.

**Solution:** Clear browser cache and check browser console for font loading errors.

### Icon Too Small/Large

**Solution:** Adjust icon size with custom CSS:

```css
.woocommerce-MyAccount-navigation a i.dashicons {
    font-size: 18px !important;
    width: 18px !important;
    height: 18px !important;
}
```

### Icons Misaligned with Text

**Solution:**
```css
.woocommerce-MyAccount-navigation a i.dashicons {
    vertical-align: middle;
    margin-right: 6px;
}
```

## Accessibility Considerations

<Info>
  Icons are decorative and don't convey information not already present in the text label, so they don't require additional ARIA labels.
</Info>

If you want to add screen reader support:

```javascript
// Add aria-hidden to icons
jQuery('.woocommerce-MyAccount-navigation a i.dashicons').attr('aria-hidden', 'true');
```

## Best Practices

<Tip>
  **Consistency:** Use similar icon styles (all filled or all outline) for a cohesive look.
</Tip>

<Tip>
  **Relevance:** Choose icons that clearly represent the menu item's purpose (e.g., cart icon for Orders).
</Tip>

<Tip>
  **Contrast:** Ensure icon colors have sufficient contrast against the background for accessibility.
</Tip>

<Warning>
  **Don't Overdo It:** Not every item needs an icon. Use them strategically to highlight important sections.
</Warning>
